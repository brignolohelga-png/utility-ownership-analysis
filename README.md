# Utility Ownership Analysis

## Research Question

Do residential electricity rates differ across utility ownership types in the United States?

## Dataset

- U.S. Electric Utility Companies and Rates: Look-up by Zip Code (2020)

## Methods

- Data cleaning and merging
- Descriptive statistics
- Boxplot visualization
- One-way ANOVA
- Tukey post-hoc test

## Main Finding

Residential electricity rates differ significantly across utility ownership types (one-way ANOVA, p < .001). Tukey's post-hoc test identified which ownership types differed significantly. Federal utilities have the lowest residential rates
